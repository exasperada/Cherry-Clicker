# Cherry Clicker

Jogo incremental desenvolvido com HTML, CSS e JavaScript, inspirado em jogos do estilo Clicker. O objetivo é acumular cerejas através de cliques e adquirir melhorias para aumentar a quantidade de cerejas obtidas por clique.

## Sobre o Projeto

Cherry Clicker é um jogo simples onde o jogador clica em uma cereja para acumular pontos (cerejas). Conforme progride, é possível comprar upgrades que aumentam a quantidade de cerejas recebidas a cada clique, permitindo uma evolução mais rápida dentro do jogo.

O projeto foi desenvolvido para praticar conceitos de desenvolvimento web, manipulação do DOM e lógica de programação com JavaScript.

## Funcionalidades

* Tela inicial interativa com efeito hover na cereja;
* Sistema de cliques para acumular cerejas;
* Contador de cerejas atualizado em tempo real;
* Exibição da quantidade de cerejas obtidas por clique;
* Sistema de upgrades com diferentes níveis de evolução;
* Compra de melhorias utilizando as cerejas acumuladas;
* Validação para impedir compras sem saldo suficiente;
* Efeito visual ao clicar na cereja;
* Contador de upgrades adquiridos;
* Condição de vitória ao atingir 500 cerejas;
* Reinicialização automática do jogo após a vitória.


## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Manipulação do DOM

## Como Jogar

1. Acesse a página inicial.
2. Clique na cereja para iniciar o jogo.
3. Clique na cereja principal para acumular pontos.
4. Compre upgrades para aumentar suas cerejas por clique.
5. Continue evoluindo para obter cada vez mais cerejas.
